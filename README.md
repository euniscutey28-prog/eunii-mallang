# 🌸 이은이의 말랑이 월드 (Eunii's Mallang World)

귀엽고 쫀득한 말랑이들을 직접 만들고, 만지며 스트레스를 풀고, 영리한 AI 상인과 교환 거래를 즐길 수 있는 인터랙티브 웹 애플리케이션입니다.

👉 **프로젝트 링크:** [GitHub 저장소 바로가기](https://github.com/euniscutey28-prog/eunii-mallang)

---

## ✨ 주요 기능 안내

### 1. 🔮 말랑이 쫀득 시뮬레이터 (Simulator)
*   **쫀득한 물리 엔진**: 클릭(꾹 누르기)하거나 드래그하는 움직임에 맞춰 말랑이가 탄력있게 찌그러지고 길게 늘어납니다. 마우스를 놓으면 감쇠 오실레이션(Spring-Mass Damper) 물리 공식을 통해 원래 모양으로 쫀득하게 바운스되며 복원됩니다.
*   **리얼 사운드**: 외부 오디오 에셋 다운로드 없이 브라우저 내장 **Web Audio API**를 사용해 실시간으로 "뾱뾱" 거리는 귀여운 힐링 효과음을 직접 합성하여 출력합니다.
*   **표정 변화**: 클릭하거나 터치하면 찌그러지며 `>_<` 모양의 리얼한 표정으로 변합니다.

### 2. ⚖️ 말랑이 밀당 거래 게임 (Trading Game)
*   **AI 상인과 거래**: 기저귀를 찬 귀여운 상인이 내 제안을 평가합니다. 상인의 눈치와 표정이 거래의 공정성에 따라 실시간으로 변합니다. (웃음 😊, 당황 😅, 분노 😡)
*   **가치 평가 물리 저울**: 나와 상인이 제안한 말랑이들의 등급별 가치(Common, Rare, Epic, Legendary)를 합산하여 저울 바가 실시간으로 기울어집니다.
*   **밀당의 즐거움**: `더 달라고 요구 (+)` 버튼을 누르면 AI가 손익을 따져 자신의 소장용 보관함에서 여분의 말랑이를 꺼내 거래판에 추가로 올려줍니다. 너무 억지스러운 딜일 경우 머리를 흔들며 거절합니다.
*   **거래 완료**: 쌍방이 승인하면 폭죽과 함께 거래가 완료되어 보관함(LocalStorage)에 영구 획득됩니다.

### 3. 🎨 나만의 말랑이 만들기 (DIY Creator)
*   **커스터마이징**: 6종의 기본 폼(원형, 구름, 곰, 하트, 별, 고양이)을 기반으로, 그라데이션 색상 배합, 6가지 깜찍한 눈코입 표정, 5가지 아기자기한 장식품(왕관, 리본, 썬글라스, 마법모자, 머리핀)을 조합할 수 있습니다.
*   **이름 등록**: 나만의 시그니처 이름을 붙여 도감에 탄생시키면 도감에 추가되며 시뮬레이터와 거래에서 즉시 사용 가능합니다.

## 🛠️ 기술 스택 및 구조
*   **Frontend**: HTML5, Vanilla JavaScript (ES6+), Vanilla CSS3 (Custom Variables, Flexbox, Keyframe Animations)
*   **Graphics**: XML-based Dynamic Inline SVG (Scalable & Light-weight)
*   **Audio Engine**: Web Audio API Synthesizer (`sound.js`)
*   **Data Persistence**: HTML5 LocalStorage API (컬렉션 영구 저장)

---

## 🚀 로컬에서 실행하는 방법

1. 저장소를 클론받거나 파일을 다운로드합니다.
2. 터미널을 열고 해당 폴더로 이동합니다.
3. 파이썬 웹 서버를 작동시킵니다:
   ```bash
   python3 -m http.server 8000
   ```
4. 크롬 등 브라우저를 켜고 아래 주소로 접속합니다:
   ```
   http://localhost:8000
   ```
